## <a name="explicit-configuration"></a><span data-ttu-id="b0cbd-107">Явной настройки</span><span class="sxs-lookup"><span data-stu-id="b0cbd-107">Explicit configuration</span></span>

<span data-ttu-id="b0cbd-108">Объект, к которому типы никогда не включаются ядром EF в модели по соглашению, которым владеет.</span><span class="sxs-lookup"><span data-stu-id="b0cbd-108">Owned entity types are never included by EF Core in the model by convention.</span></span> <span data-ttu-id="b0cbd-109">Можно использовать `OwnsOne` метод в `OnModelCreating` или указать для типа `OwnedAttribute` (новые возможности EF Core 2.1) чтобы настроить тип как тип владельца.</span><span class="sxs-lookup"><span data-stu-id="b0cbd-109">You can use the `OwnsOne` method in `OnModelCreating` or annotate the type with `OwnedAttribute` (new in EF Core 2.1) to configure the type as an owned type.</span></span>

<span data-ttu-id="b0cbd-110">В этом примере StreetAddress — это тип с нет свойства identity.</span><span class="sxs-lookup"><span data-stu-id="b0cbd-110">In this example, StreetAddress is a type with no identity property.</span></span> <span data-ttu-id="b0cbd-111">Он используется как свойство сущности Order для указания адреса доставки конкретного заказа.</span><span class="sxs-lookup"><span data-stu-id="b0cbd-111">It is used as a property of the Order type to specify the shipping address for a particular order.</span></span> <span data-ttu-id="b0cbd-112">В `OnModelCreating`, мы используем `OwnsOne` метод, чтобы указать, что свойство ShippingAddress является принадлежащий сущности типа Order.</span><span class="sxs-lookup"><span data-stu-id="b0cbd-112">In `OnModelCreating`, we use the `OwnsOne` method to specify that the ShippingAddress property is an Owned Entity of the Order type.</span></span>
